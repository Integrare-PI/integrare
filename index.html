A Integrare

Baseada na ODS 4 da ONU, a Integrare é uma rede social de perguntas e resposta e permite que o aluno se conecte e compartilhe suas dúvidas com educadores especializados.

Clique aqui para conhecer: Integrare

Front End

Tecnologias utilizadas: HTML, CSS, TypeScript, React, Material UI
Acesso ao código aqui: GitHub

Back End

Tecnologias utilizadas: Java, Spring Boot, Spring Web, Spring Data JPA, MySQL
